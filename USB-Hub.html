USB Hub


Introduction:

Have you ever wanted extra USB Ports on your computer? Maybe you have a Macbook and only have a couple USB-C ports. There is a large market for USB hubs that expand a single USB port into many. With the introduction of USB-C and USB 3.0 it is possible to that single USB port into multiple HDMI ports as well. You can find many different types of these devices at any electronics retailer, but if you want to make your own you have come to the right place. The following is a detailed overview of how to create a USB 3.0 hub turning one USB port into four. This version of the project will not include any HDMI or video ports it will focus on expanding a USB 3.0 port.
