USB3 Hub


Introduction:

Have you ever wanted extra USB Ports on your computer? Maybe you have a Macbook and only have a couple USB-C ports. There is a large market for USB hubs that expand a single USB port into many. With the introduction of USB-C and USB 3.0 it is possible to that single USB port into multiple HDMI ports as well. You can find many different types of these devices at any electronics retailer, but if you want to make your own you have come to the right place. The following is a detailed overview of how to create a USB 3.0 hub turning one USB port into four. This version of the project will not include any HDMI or video ports it will focus on expanding a USB 3.0 port. This USB Hub design is going to revolve around the USB5744/2G IC [1] and the schematic and PCB design will be done in KiCad [2], a free ECAD tool. All supporting documents and design files can be found in my github repo [3].

The objective is to design a hub that can expand one USB3.0 into four on any PC the user of would like, see figure 1. USB type-c has been quite fashionable in recent years, type-c also has all the necessary pins to support USB3.0. We will use a type-c receptacle as the upstream port on our hub. This means the user will need a cable which supports USB3.0 and has a type-c plug on one end and a plug on the other end which is compatible with the host PC. Most hubs you see at an electronics retailers have there own cable attached to the hub. This is because not all USB cables are made the same, they want to control the impedance and power requirements of the cable. This hub will rely on the user of the to make sure they have a working USB3 cable. The following will give an overview of the schematic and printed circuit board (PCB) layout, if you would like to read more about this subject I suggest looking at the sources and attempting to design a Hub yourself.
